# Vosk Speech-to-Text Models

This directory contains models for Vosk speech recognition engine.

## Setup

1. Install Vosk:
```bash
npm install vosk
```

2. Download models from: https://alphacephei.com/vosk/models

3. Extract model files to this directory

## Supported Languages
- English (en)
- Hindi (hi)
- And many more...

## Usage

Update `useSTT.js` hook to load models from this directory.
